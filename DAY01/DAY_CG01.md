# Day 1 – Introduction to Linux (DevOps Foundation)

### 1️⃣ Concept (30 min)

Most DevOps servers run on **Linux**, not Windows.

Examples of Linux distributions:

* Ubuntu
* CentOS
* RedHat
* Debian
* Amazon Linux

Why DevOps engineers use Linux:

* Open source
* Lightweight
* Secure
* Powerful command line
* Easy automation

Typical DevOps workflow:

Developer → Git → CI/CD → Docker → Kubernetes → Cloud

Linux manages servers in this pipeline.

# 3️⃣ Basic Linux Commands (Practice – 1 hour)

Open the terminal and practice these commands.

### Check current directory

```bash
pwd
```

---

### List files

```bash
ls
```

Show hidden files

```bash
ls -a
```

---

### Change directory

```bash
cd foldername
```

Go back

```bash
cd ..
```

---

### Create folder

```bash
mkdir devops
```

Go inside folder

```bash
cd devops
```

---

### Create file

```bash
touch file1.txt
```

---

### Show file content

```bash
cat file1.txt
```

---

### Clear terminal

```bash
clear
```
